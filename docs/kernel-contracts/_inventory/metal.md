# Metal kernel inventory — `fuel-metal-kernels` + `fuel-metal-backend`

Scope: every distinct kernel/op that Fuel itself provides through `fuel-metal-kernels`
(Metal `.metal` sources + the Rust `call_*` dispatch wrappers) and how `fuel-metal-backend`
(`storage.rs`) wires them. dtype-monomorphized variants of one kernel are listed as a dtype
list on a single entry. The GGML quantized `.metal` file ships hundreds of kernels but only the
two `call_quantized_matmul_*` entry points are reachable through Fuel, so only those are listed.

Cross-cutting facts:
- Every contiguous host-name kernel comes paired with a `_strided` variant generated by the same
  `init_*` macro. The Rust wrappers (`call_*_contiguous` vs `call_*_strided`) pick the variant; the
  backend `storage.rs` chooses based on `layout.is_contiguous()`.
- Element/byte offsets are handled by `BufferOffset { buffer, offset_in_bytes }`
  (`utils.rs`), set via `set_buffer(pos, buf, offset)`. So all `call_*` that take a `BufferOffset`
  are **non-zero-offset capable** for their inputs. The backend computes
  `offset_in_bytes = layout.start_offset() * dtype.size_in_bytes()` (`buffer_o`).
- The strided indexer `get_strided_index` (in unary/binary/affine/cast/ternary/indexing metal)
  is the generic mixed-radix de-linearizer; it makes those kernels accept **arbitrary strides
  including broadcast (stride 0) and transposed/overlapping layouts**, bounded only by `uint`
  index range.
- Contiguous kernels use `work_per_thread<T>() = ceil(8/sizeof(T))` tiling
  (`get_tile_size`: f32→2, f16/bf16→4, u8→8); strided kernels are one-thread-per-element.
- Output buffers are freshly allocated (`device.new_buffer`) and treated as contiguous unless the
  op writes through a `BufferOffset` dst (copy_strided / const_set strided / unary strided).

## Unary (`unary.metal`, `kernels/unary.rs`)

### `unary_kernel` / `unary_kernel_strided` (elementwise unary)
- **Op kind:** elementwise unary map `out[i] = op(in[i])`.
- **Ops (one struct each):** cos, sin, exp, sqr, sqrt, neg, log, gelu (tanh-approx), gelu_erf,
  abs, ceil, floor, relu, round, erf, recip, silu, sign, sigmoid, tanh, copy (`uid`), const_set.
  (Note: `ops!` macro also declares badd-style names for symmetry but unary uses only the unary set.)
- **dtypes:** float family f32/f16/bf16 for all math ops (`init_unary_float`). copy adds u8/u32/i64.
  Integer math ops are NOT generated. Backend `unary()` only wires f32/f16/bf16 (plus `sign` i64).
- **Input layouts:** contiguous variant requires dense layout (length only). strided variant accepts
  **arbitrary strides incl. broadcast / non-contiguous** via `get_strided_index`. Offset-capable (BufferOffset).
- **Output:** same dtype as input, same shape; contiguous fresh buffer. Strided variant writes
  `out[tid]` densely (gather-from-strided-input → contiguous output), and the backend passes a
  `BufferOffset` dst so it is also offset-capable (used by `copy_strided_src`).
- **Precision:** f32-internal in several ops; gelu uses `precise::tanh`; erf is A&S 7.1.26 polynomial
  (NOT exact erf); abs casts to float. NOTE: a dead `relu` METAL_FUNC duplicates gelu but the wired
  `relu` host kernel maps to `urelu` (`x<0?0:x`), which is correct.
- **Source:** `metal_src/unary.metal:37-66,168-278`; `kernels/unary.rs:16-68`.

### `const_set` / `const_set_strided`
- **Op kind:** fill an existing buffer region with a scalar (writes `out[i]=value`).
- **dtypes:** f32/f16/bf16/u8/u32/i64.
- **Input layouts:** scalar input by value; contiguous fills `[0,length)`. strided writes
  `out[get_strided_index(tid)]` → scatters into a strided dst (offset-capable dst).
- **Output:** in-place on the supplied dst buffer (aliasing: mutates caller's storage). dtype = dst dtype.
- **Source:** `metal_src/unary.metal:68-95`; `kernels/unary.rs:70-120`; backend `const_set` `storage.rs:449-541`.

### `copy2d` (`copy2d_<t>`)
- **Op kind:** 2-D strided block copy: `out[x*dst_s + y] = in[x*src_s + y]` over a `d1×d2` grid.
- **dtypes:** f32/f16/bf16/u8/u32/i64.
- **Input layouts:** explicit `(d1,d2,src_s,dst_s)` + byte offsets on src and dst; supports
  row-strided 2-D source and dest. Not a general N-D strider — exactly one stride pair.
- **Output:** writes into caller dst at `dst_o_in_bytes` (offset/alias capable). dtype = src dtype.
- **Notes:** Used by `copy2d()` (falls back to blit when `src_s==d2==dst_s`) and by mlx multi-block sort.
- **Source:** `metal_src/unary.metal:97-111`; `kernels/unary.rs:122-173`; backend `storage.rs:1763-1821`.

## Binary (`binary.metal`, `kernels/binary.rs`)

### `binary_kernel` + `binary_kernel_strided` (with `_lstrided`/`_rstrided` indexer variants)
- **Op kind:** elementwise binary `out[i] = op(l[i], r[i])`.
- **Ops:** arithmetic badd, bsub, bmul, bdiv, bminimum, bmaximum (return T);
  comparisons eq, ne, le, lt, ge, gt (`init_boolean_binary`, return **bool→u8**).
- **dtypes:** f32/f16/bf16/u8/u32/i64 for both arithmetic and comparison.
- **Input layouts:** contiguous (`_`) needs both dense. Strided family has FOUR monomorphizations per
  op/dtype selected by the backend: `_strided` (both strided), `_lstrided` (left strided, right
  contiguous), `_rstrided` (left contiguous, right strided). All use `get_strided_index` → accept
  **arbitrary/broadcast strides**. Offset-capable on both inputs.
- **Output:** arithmetic→input dtype; comparison→u8 (`beq`… return bool, backend sets out dtype U8).
  Same shape (driven by lhs shape/dims), contiguous fresh buffer.
- **Precision:** plain operators; min/max via MIN/MAX macro (NaN-naive, picks by `>`/`<`).
- **Source:** `metal_src/binary.metal:61-196`; `kernels/binary.rs`; backend `binary()` `storage.rs:1889-1957`,
  `cmp()` `storage.rs:437-447`.

## Affine / powf / elu (`affine.metal`, `kernels/affine.rs`)

### `affine_kernel` / `_strided`
- **Op kind:** fused multiply-add `out = in*mul + add` (`fma(float(in),mul,add)`).
- **dtypes:** u8/u32/i64/f32/f16/bf16 (integers only via contiguous+strided; backend wires all six).
- **Input layouts:** contiguous (dense, tiled) or strided (`get_strided_index`, broadcast-capable, offset-capable).
- **Output:** same dtype/shape; strided variant reads strided, writes `out[tid]` dense.
- **Precision:** accumulates in f32 then casts back (so f16/bf16 affine is f32-internal).
- **Source:** `metal_src/affine.metal:36-68`; `kernels/affine.rs:6-75`; backend `affine()` `storage.rs:122-183`.

### `powf_kernel` / `_strided`
- **Op kind:** `out = pow(float(in), mul)`.
- **dtypes:** f32/f16/bf16 only.
- **Input layouts:** contiguous / strided (broadcast-capable, offset-capable).
- **Output:** same dtype/shape. f32-internal pow then cast.
- **Source:** `metal_src/affine.metal:70-99`; `kernels/affine.rs:77-135`; backend `powf()` `storage.rs:185-238`.

### `elu_kernel` / `_strided`
- **Op kind:** ELU `out = x>0 ? x : mul*(exp(x)-1)`.
- **dtypes:** f32/f16/bf16.
- **Input layouts:** contiguous / strided (broadcast-capable, offset-capable).
- **Output:** same dtype/shape. exp computed in T (not promoted to f32 in strided/contiguous body).
- **Source:** `metal_src/affine.metal:101-132`; `kernels/affine.rs:137-195`; backend `elu()` `storage.rs:240-291`.

## Cast (`cast.metal`, `kernels/cast.rs`)

### `cast_kernel` / `cast_kernel_strided`
- **Op kind:** dtype conversion `out = U(IR(in))` (IR=intermediate, defaults to T).
- **dtypes:** all ordered pairs over {f32,f16,bf16,i64,u32,u8} (`init_cast_all` × 6 src). Backend
  `to_dtype` wires the full strided set and the contiguous set MINUS same-dtype and minus bf16-as-source
  contiguous entries that route via strided (see name tables) — effectively all cross casts.
- **Input layouts:** contiguous (dense) or strided (`get_strided_index`, broadcast/offset-capable).
- **Output:** dtype = target U; same shape; contiguous fresh buffer. Strided reads strided, writes dense.
- **Precision:** straight `static_cast` chain; no rounding control. Note: `IR` template param is unused
  by the dispatch (always = T), so no special int-truncation intermediate is requested.
- **Source:** `metal_src/cast.metal:36-104`; `kernels/cast.rs`; backend `to_dtype()` `storage.rs:543-663`.

## Reduce / norm / rope (`reduce.metal`, `kernels/reduce.rs`)

### `fast_<op>_<t>` / `_strided` (block reductions)
- **Op kind:** axis reduction collapsing the trailing (sum) dims into one output element per group.
  Ops: Sum, Mul, Max, Min (value-returning); ArgMax, ArgMin (index-returning, `impl_arg_reduce`).
- **dtypes:** f32/f16/u32/u8 always; +bf16 and +i64 (guarded). Softmax-style MD path is f32-accum.
- **Input layouts:** contiguous variant indexes linearly (`indexer_t<uint,false>`). `_strided` variant
  uses a templated `strided_indexer` (specialized for rank 1-4, fallback loop for >4) → accepts
  **arbitrary strides** over the reduced region. Offset-capable (BufferOffset). The backend only takes
  the strided path when the reduction sub-shape is non-contiguous; otherwise contiguous.
- **op params:** `src_numel`, `num_dims`, `dims[]` (+`strides[]` strided), `el_per_block`.
- **Output:** value ops → input dtype; arg ops → **u32 index** (`storer<indexed<T>>` writes `.i`).
  Output shape = src with reduced dims removed (one element per threadgroup). Backend errors on empty
  tensor for min/max/arg. Tie-break: arg ops keep lowest index (`<` includes `i<i`).
- **Precision:** Sum/Mul/Min/Max in T (Min/Max use `fast::min/max` for float). simd-shuffle tree reduce.
- **Source:** `metal_src/reduce.metal:578-765,1502-1543`; `kernels/reduce.rs:6-117`; backend `reduce_op()` `storage.rs:293-435`.

### `softmax_<t>` (last-dim online softmax)
- **Op kind:** softmax over the last `el_per_block` elements (Welford/online normalizer, MD struct).
- **dtypes:** f32/f16/bf16.
- **Input layouts:** **contiguous only** — `indexer_t<uint,false>`, no strided variant; backend passes
  raw `(input, input_offset)` (offset-capable but dense). Output written dense.
- **op params:** `src_numel`, `el_per_block` (= last-dim length).
- **Output:** same dtype/shape; `d` accumulator is f32 (so normalization is f32-internal).
- **Source:** `metal_src/reduce.metal:848-933`; `kernels/reduce.rs:119-165` (`call_last_softmax`).
- **Note:** no wired consumer found in `fuel-metal-backend` (entry point exists in kernels crate).

### `rmsnorm_<t>`
- **Op kind:** RMS norm `out = x * rsqrt(mean(x^2)+eps) [* alpha]`.
- **dtypes:** f32/f16/bf16.
- **Input layouts:** contiguous only (`indexer_t<uint,false>`); src/alpha offset-capable via BufferOffset.
  `alpha` optional (nullptr → skip scale); when present it is read per-block-local (`alpha[i-offset]`)
  so alpha length == el_per_block.
- **op params:** `length`, `elements_to_sum` (=last dim), `eps`, alpha buffer+offset.
- **Output:** same dtype/shape; reduction/accumulation in **f32** (RMS<float>).
- **Source:** `metal_src/reduce.metal:1058-1160`; `kernels/reduce.rs:167-223` (`call_rms_norm`).

### `layernorm_<t>`
- **Op kind:** LayerNorm `out = (x-mu)*rsqrt(var+eps); *alpha +beta` (alpha/beta optional, null-guarded).
- **dtypes:** f32/f16/bf16.
- **Input layouts:** contiguous only; src/alpha/beta offset-capable; alpha/beta block-local indexed.
- **op params:** `length`, `elements_to_sum`, `eps`, alpha+offset, beta+offset.
- **Output:** same dtype/shape; mean/m2 in **f32** (LayerNormValue<float>).
- **Source:** `metal_src/reduce.metal:1241-1363`; `kernels/reduce.rs:225-286` (`call_layer_norm`).

### `rope_i_<t>` / `rope_<t>` / `rope_thd_<t>` (rotary embeddings)
- **Op kind:** RoPE. Three variants: interleaved (`ropei`), default split-half (`rope`),
  and (b,t,h,d) layout (`rope_thd`). Per-pair rotate using cos/sin tables, optional batch stride.
- **dtypes:** f32/f16/bf16.
- **Input layouts:** **contiguous, fixed dim arithmetic** (no general strider; uses bh/td/d/stride_b
  index math). src/cos/sin/out all offset-capable via BufferOffset. `stride_b>0` enables per-batch cos/sin.
- **op params (per variant):** ropei: bh, td, stride_b; rope: bh, td, d, stride_b; rope_thd: b, t, h, d, stride_b.
- **Output:** same dtype/shape; arithmetic in T.
- **Source:** `metal_src/reduce.metal:1365-1500`; `kernels/reduce.rs:288-421` (`call_rope_i`/`call_rope_thd`/`call_rope`).
- **Note:** wired by retired `_fuel_nn_retired/rotary_emb.rs`, not by the live backend.

## Indexing (`indexing.metal`, `kernels/indexing.rs`)

### `is_<idx>_<t>` (index_select / embedding gather along `dim`)
- **Op kind:** `output[tid] = input[ left*dimsz*right + id*right + right ]`, id clamped to dim-1;
  sentinel max-value id → writes 0.
- **dtypes (idx × val):** idx ∈ {i64,u32,u8}; val ∈ {u8,u32,i64,f32,f16,bf16} (subset per idx; see
  backend match `storage.rs:1606-1633`).
- **Input layouts:** **ids must be contiguous** (backend enforces). Source supports a `contiguous` bool:
  if false, source index is remapped via `get_strided_index(src_i, src_dims, src_strides)` →
  source may be **strided/non-contiguous**. Offset-capable on input and ids.
- **op params:** dst_el, left_size, src_dim_size, right_size, ids_size, contiguous, src_dims[], src_strides[], dim.
- **Output:** val dtype; shape = ids shape with src dims around `dim`; contiguous fresh buffer.
- **Source:** `metal_src/indexing.metal:37-89,262-285`; `kernels/indexing.rs:6-58`; backend `index_select()` `storage.rs:1589-1654`.

### `gather_<idx>_<t>` (gather along `dim`, full index tensor)
- **Op kind:** `output[tid] = input[(left*dimsz + ids[tid])*right + right]`; sentinel id → 0.
- **dtypes (idx × val):** idx ∈ {u8,u32,i64}; val ∈ {f32,f16,bf16,u8,u32,i64} (combinations in metal 287-304).
- **Input layouts:** **ids contiguous required** (backend). Source addressed with implied contiguous
  layout (NO strided remap inside gather) — source effectively **contiguous-assumed**; offset-capable.
- **op params:** dst_el, left_size, src_dim_size, right_size, ids_size, dim.
- **Output:** val dtype; shape = ids shape; contiguous fresh buffer.
- **Source:** `metal_src/indexing.metal:92-131,287-304`; `kernels/indexing.rs:60-106`; backend `gather()` `storage.rs:1439-1487`.

### `s_<idx>_<t>` (scatter, set) and `sa_<idx>_<t>` (scatter_add)
- **Op kind:** scatter: `out[(left*dst_dimsz+idx)*right+r] = in[src_i]`; scatter_add: `+= in[src_i]`.
  Loops over `src_dim_size`; ids ≥ max-value sentinel are skipped.
- **dtypes (idx × val):** idx ∈ {u8,u32,i64}; val ∈ {f32,f16,bf16} (+u32/u32). Backend match `storage.rs:1501-1517` / `1551-1567`.
- **Input layouts:** **all of dst (`l`), ids, src must be contiguous** (backend enforces
  RequiresContiguous). Offsets via BufferOffset still applied. Implicit row-major addressing inside kernel.
- **op params:** dst_el(=left*right), left_size, src_dim_size, right_size, dst_dim_size, dim.
- **Output:** in-place into the dst storage (aliasing — `scatter_set`/`scatter_add_set` mutate `self`).
  dtype = dst dtype. scatter_add does non-atomic `+=` (race-prone if duplicate targets across threads;
  here each `tid` owns a (left,right) column so writes are disjoint).
  NOTE: `scatter_add_set` backend calls `call_scatter` (the SET kernel `sa_*` passed in as name) — the
  name string is the `sa_*` add kernel, dispatched through the generic `call_scatter` wrapper.
- **Source:** `metal_src/indexing.metal:133-215,306-330`; `kernels/indexing.rs:108-155`; backend `storage.rs:1489-1587`.

### `ia_<idx>_<t>` (index_add)
- **Op kind:** `out[(left*dst_dimsz+ids[j])*right+r] += in[(left*src_dimsz+j)*right+r]` over `ids_dim_size`.
- **dtypes (idx × val):** idx ∈ {i64,u32,u8}; val ∈ {f16,f32,i64,u32,u8,bf16} (full per backend `storage.rs:1670-1697`).
- **Input layouts:** ids and src **must be contiguous** (backend); destination is a pre-zeroed/copied
  accumulator (`copy_strided_src` of `l` into fresh buffer). Offset-capable.
- **op params:** dst_el, left_size, src_dim_size, right_size, dst_dim_size, ids_dim_size, dim.
- **Output:** new buffer (accumulator), dtype = self dtype; non-atomic `+=` (disjoint columns).
- **Source:** `metal_src/indexing.metal:217-259,332-360`; `kernels/indexing.rs:157-206`; backend `index_add()` `storage.rs:1656-1717`.

## Ternary (`ternary.metal`, `kernels/ternary.rs`)

### `where_<idcond>_<t>` (where_cond / select)
- **Op kind:** `out[i] = select(f[f_idx], t[t_idx], cond[idx])`.
- **dtypes:** cond ∈ {u8,u32,i64}; value T ∈ {f16,f32,u8,u32,i64,bf16}. Backend only wires
  cond=u8/u32 with T ∈ {f32,f16,bf16,i64,u32,u8} (`storage.rs:868-879`).
- **Input layouts:** each of cond/t/f independently contiguous OR strided — driven by 3 boolean
  **function constants** (IDS_CONTIGUOUS / T_CONTIGUOUS / F_CONTIGUOUS) set at pipeline-compile time;
  strided uses `get_strided_index` → broadcast/non-contig capable per-operand. Offset-capable on all three.
- **op params:** numel, num_dims, dims[], cond_stride[], t_stride[], f_stride[] + the 3 contiguity bools.
- **Output:** value dtype T; shape from t layout; contiguous fresh buffer.
- **Source:** `metal_src/ternary.metal:40-126`; `kernels/ternary.rs`; backend `where_cond()` `storage.rs:845-903`.

## Fill (`fill.metal`, `kernels/fill.rs`)

### `fill_<t>` (const fill)
- **Op kind:** `out[tid] = value` for `tid<numel`.
- **dtypes:** u8/u32/i64/f16/f32/bf16.
- **Input layouts:** dense output only, no offset/stride parameters (writes `[0,numel)` of the buffer).
- **Output:** writes into supplied buffer; dtype = T; in-place/aliasing on caller buffer.
- **Source:** `metal_src/fill.metal`; `kernels/fill.rs` (`call_const_fill`).
- **Note:** distinct from `const_set` (unary): no offset, no strided variant.

## Convolution / pooling / upsample (`conv.metal`, `kernels/convolution.rs`)

### `im2col_<t>` (2-D im2col)
- **Op kind:** im2col for conv2d: dst `(b,h_out,w_out,c_in,h_k,w_k)` gathered from src `(b,c_in,h_in,w_in)`,
  zero-padded out of range.
- **dtypes:** f32/f16/bf16/u8/u32.
- **Input layouts:** **strided source** — reads via `src_strides[0..4]` so input may be non-contiguous;
  offset-capable. Output dense.
- **op params:** dst_numel, h_out, w_out, h_k, w_k, stride, padding, dilation, src_dims[], src_strides[].
- **Output:** same dtype; contiguous fresh buffer in im2col layout. (Backend then matmuls + copies.)
- **Source:** `metal_src/conv.metal:7-69,649-655`; `kernels/convolution.rs:69-104`; backend `conv2d()` `storage.rs:1099-1181`.

### `im2col1d_<t>` (1-D im2col)
- **Op kind:** im2col for conv1d: dst `(b,l_out,c_in,l_k)` from src `(b,c_in,l_in)`, zero-padded.
- **dtypes:** f32/f16/bf16/u8/u32.
- **Input layouts:** strided source (`src_strides[0..3]`), offset-capable; dense output.
- **op params:** dst_numel, l_out, l_k, stride, padding, dilation, src_dims[], src_strides[].
- **Output:** same dtype; contiguous fresh buffer.
- **Source:** `metal_src/conv.metal:116-159,665-671`; `kernels/convolution.rs:6-34`; backend `conv1d()` `storage.rs:905-980`.

### `col2im1d_<t>` (col2im for conv_transpose1d)
- **Op kind:** scatter-accumulate columns back to image: dst `(b,c_out,l_out)` from src `(b,l_in,c_out,l_k)`.
- **dtypes:** f32/f16/bf16/u8/u32.
- **Input layouts:** **contiguous assumed** — internal strides computed from shape, not passed in;
  offset-capable on src (zero-offset in practice). Output zero-init then `+=`.
- **op params:** dst_el, l_out, l_in, c_out, k_size, stride.
- **Output:** same dtype; contiguous fresh buffer.
- **Source:** `metal_src/conv.metal:71-113,657-663`; `kernels/convolution.rs:36-66`; backend `conv_transpose1d()` `storage.rs:982-1097`.

### `upsample_nearest2d_<t>`
- **Op kind:** nearest-neighbor 2-D upsample over NCHW (treated as b,c,w,h here).
- **dtypes:** f32/f16/bf16/u8/u32.
- **Input layouts:** strided source (`src_s[0..4]`), offset-capable; dense output.
- **op params:** w_out, h_out, w_scale, h_scale (= in/out ratios), src_dims[], src_strides[].
- **Output:** same dtype; contiguous fresh buffer.
- **Source:** `metal_src/conv.metal:161-200,673-679`; `kernels/convolution.rs:106-135`; backend `upsample_nearest2d()` `storage.rs:1342-1380`.

### `upsample_bilinear2d_<t>`
- **Op kind:** bilinear 2-D upsample (PyTorch align_corners + scale semantics).
- **dtypes:** f32/f16/bf16/u8/u32.
- **Input layouts:** strided source (`src_s[0..4]`), offset-capable; dense output (NCHW).
- **op params:** w_out, h_out, align_corners, has_scale_h, scale_h_factor, has_scale_w, scale_w_factor,
  src_dims[], src_strides[]. (Explicit `[[buffer(N)]]` bindings.)
- **Output:** same dtype; interpolation in **f32** then cast. Contiguous fresh buffer.
- **Source:** `metal_src/conv.metal:202-284,681-687`; `kernels/convolution.rs:137-182`; backend `upsample_bilinear2d()` `storage.rs:1382-1437`.

### `max_pool2d_<t>`
- **Op kind:** 2-D max pooling.
- **dtypes:** f32/f16/bf16/u32/u8.
- **Input layouts:** strided source (`src_strides[0..4]`); offset NOT applied (backend passes raw
  `&self.buffer`, so **zero-offset only**). Dense output.
- **op params:** w_k, h_k, w_stride, h_stride, src_dims[], src_strides[].
- **Output:** same dtype; contiguous fresh buffer; accumulation in T.
- **Source:** `metal_src/conv.metal:431-497,689-695`; `kernels/convolution.rs:184-215`; backend `max_pool2d()` `storage.rs:1295-1336`.

### `avg_pool2d_<t>`
- **Op kind:** 2-D average pooling (sum/ (w_k*h_k)).
- **dtypes:** f32/f16/bf16/u32/u8 (float accumulator for f16/bf16/f32; int accum for int types).
- **Input layouts:** strided source; **zero-offset only** (backend passes raw buffer). Dense output.
- **op params:** w_k, h_k, w_stride, h_stride, src_dims[], src_strides[].
- **Output:** same dtype; accumulation A=float for float dtypes (precision: f32-accum).
- **Source:** `metal_src/conv.metal:370-429,697-703`; `kernels/convolution.rs:184-215` (shared `call_pool2d`); backend `avg_pool2d()` `storage.rs:1252-1293`.

### `conv_transpose1d_<t>`
- **Op kind:** naive transposed conv1d (per-output accumulate over kernel × c_in).
- **dtypes:** f32/f16/bf16/u8/u32 (float accumulator for floats).
- **Input layouts:** **strided** src and kernel (`src_strides`, `k_strides`), both offset-capable.
- **op params:** l_out, stride, padding, out_padding, dilation, src_dims[], src_strides[], k_dims[], k_strides[].
- **Output:** same dtype; contiguous fresh buffer; accumulation in A (f32 for floats → f32-internal).
- **Source:** `metal_src/conv.metal:501-567,705-711`; `kernels/convolution.rs:217-268`; backend `conv_transpose1d()` `storage.rs:982-1097` (non-col2im branch).

## MLX GEMM (`mlx_gemm.metal`, `kernels/mlx_gemm.rs`)

### `gemm_<trans>_<itype>_<otype>_<bm>_<bn>_<bk>_<wm>_<wn>` (steel fused GEMM)
- **Op kind:** batched matmul `D = A @ B`. trans ∈ {nn,tn,nt,tt}; tile config chosen by dtype/size/device.
- **dtypes:** f32/f16/bf16 (input == output dtype; itype==otype). Backend `matmul()` only allows these three.
- **Input layouts:** A/B must each be **row-major OR column-major in the last 2 dims** (lda/ldb derived;
  transpose detected from strides). Non-(contig|transposed) minor strides → `MatMulNonContiguous` error.
  Offsets via byte offset (offset-capable). Batch dim handled via batch_strides; **B broadcast (stride 0)
  triggers batch-collapse into M**.
- **op params:** (b,m,n,k), lhs_stride, lhs_offset, rhs_stride, rhs_offset; GemmParams struct (lda,ldb,ldd,
  tiles, batch strides, swizzle, k-iters); function constants for align_m/n/k, has_batch, gather/axpby off.
- **Output:** fresh `b*m*n` buffer, dtype = input dtype, row-major (ldd=n). No accumulate (use_out_source=false).
- **Precision:** MLX steel tiles; accumulation per kernel (typically f32 accum internally for f16/bf16).
- **Source:** `metal_src/mlx_gemm.metal` (steel templates); `kernels/mlx_gemm.rs:255-455`; backend `matmul()` `storage.rs:1719-1761`.

## SDPA (`scaled_dot_product_attention.metal`, `kernels/sdpa.rs`)

### `steel_attention_<itype>_bq_bk_bd_wm_wn_mask<mtype>` (full / prefill attention)
- **Op kind:** full scaled-dot-product attention (tiled steel kernel), optional additive mask, optional causal.
- **dtypes:** f16/bf16/f32 (itype); mask dtype f16/bf16/f32 or none. head_dim ∈ {32,64,72,80,96,128,256,512};
  bd=512 is f16/bf16 only.
- **Input layouts:** q,k,v **contiguous** (3-stride arrays passed; expects (b,h,seq,d) shape). GQA via
  gqa_factor (q_heads/kv_heads). Offsets via byte offset. align_Q/align_K + has_mask + do_causal are
  function constants.
- **op params:** q/k/v shapes+strides+offsets, mask buffer+strides+type, scale, do_causal, AttnParams struct
  (b,h,d,ql,kl,gqa,scale,softcapping=1.0,tiling,…).
- **Output:** same dtype as itype; shape = q shape; contiguous. softcapping fixed 1.0 (disabled).
- **Source:** `metal_src/scaled_dot_product_attention.metal`; `kernels/sdpa.rs:21-247` (`call_sdpa_full`).

### `sdpa_vector_<itype>_<bk>` (single-pass vector/decode attention)
- **Op kind:** SDPA for q seq-len 1 (decode), one threadgroup per (b*h).
- **dtypes:** f16/bf16/f32; head_dim bk ∈ {32,64,96,128,256,512}.
- **Input layouts:** q,k,v contiguous; k/v use `k_stride[1]`/`v_stride[1]` (head stride) so KV may be
  laid out with a head stride. Offset-capable. `sdpa_vector_has_mask` constant = false (no mask).
- **op params:** gqa_factor, n(=kv_len), kstride, vstride, alpha(scale), softcapping.
- **Output:** same dtype; q shape; contiguous. alpha pre-divided by softcapping when capping enabled.
- **Source:** `kernels/sdpa.rs:253-360` (`call_sdpa_vector`).

### `sdpa_vector_2pass_1_*` + `sdpa_vector_2pass_2_*` (two-pass vector attention)
- **Op kind:** split-K decode attention: pass-1 partial (intermediate/sums/maxs over 32 blocks),
  pass-2 reduce → output.
- **dtypes:** f16/bf16/f32; bk ∈ {32,64,96,128,256,512}.
- **Input layouts:** q,k,v contiguous (k/v head-stride as above), offset-capable; intermediate/sums/maxs
  are scratch buffers (`SDPA_2PASS_BLOCKS=32`).
- **op params:** gqa_factor, n, kstride, vstride, alpha, softcapping (pass1); intermediate/sums/maxs (both).
- **Output:** same dtype; q shape; contiguous. Internal partials in scratch.
- **Source:** `kernels/sdpa.rs:362-547` (`call_sdpa_vector_2pass`).

## Sort (`sort.metal` + `mlx_sort.metal`, `kernels/sort.rs`)

### `asort_asc_<t>` / `asort_desc_<t>` (bitonic argsort, single block)
- **Op kind:** per-row bitonic argsort → returns **u32 index permutation** (`dst` is uint32).
- **dtypes (keys):** f32/f16/u8/u32/i64/bf16. Output indices always u32.
- **Input layouts:** rows of `ncols`; src indexed as `x + row*ncols` → **contiguous rows assumed**;
  offset-capable on src. `ncols_pad` = next pow2; padded entries sort to the end.
- **op params:** ncols, ncols_pad, nrows; threadgroup shared mem `ncols_pad*4`.
- **Output:** u32 index buffer (one perm per row), dense.
- **Source:** `metal_src/sort.metal`; `kernels/sort.rs:6-41` (`call_arg_sort`). Used by `fuel-core/src/sort.rs`.

### `carg_block_sort_*` / `sort_mbsort_*` / `partition_mbsort_*` / `merge_mbsort_*` (MLX multi-block argsort)
- **Op kind:** MLX block / multi-block argsort returning u32 indices; auto-selects single-block
  (`block_sort`) vs multi-block merge pipeline (`multi_block_sort`) by column count.
- **dtypes (keys):** uint8/uint32/int64/float16/bfloat16/float32 (`mlx_dtype_str`); idx uint32.
- **Input layouts:** contiguous rows (`stride_sorted_axis=1`); src offset-capable. Multi-block allocates
  ping-pong val/idx scratch + block_partitions; final copy via `copy2d` with row strides.
- **op params:** bn, tn (tn=8; bn∈{128,256,512} by ncols), nblocks, nrows, ncols; merge tiles loop.
- **Output:** u32 index buffer, row-major; copied to dst via copy2d.
- **Source:** `metal_src/mlx_sort.metal`; `kernels/sort.rs:54-292` (`call_mlx_arg_sort`).

## Random (`random.metal`, `kernels/random.rs`)

### `rand_uniform_<t>`
- **Op kind:** fill buffer with U(min,max) via HybridTaus (Tausworthe+LCG), mirror-fill symmetry.
- **dtypes:** f32/f16/bf16.
- **Input layouts:** output dense `[0,size)`; per-thread handles 2 elements (tid and `size-off-tid`).
  Seed buffer is an atomic `seed[2]` read+written (thread 0 advances seed). No input tensor.
- **op params:** size, min, max, seed buffer. (Backend errors if min>=max.)
- **Output:** dtype = T, dense; values `rng*|max-min|+min`. RNG math in f32 then cast.
- **Source:** `metal_src/random.metal:132-208`; `kernels/random.rs:6-38`; backend `rand_uniform()` `storage.rs:2087-2122`.

- **Op kind:** fill with N(mean,stddev) via Box-Muller (two values per thread), mirror-fill.
- **dtypes:** f32/f16/bf16.
- **Input layouts:** output dense; atomic seed buffer advanced by thread 0. No input tensor.
- **op params:** size, mean, stddev, seed buffer.
- **Output:** dtype = T, dense; f32-internal then cast.
- **Source:** `metal_src/random.metal:162-220`; `kernels/random.rs:40-67`; backend `rand_normal()` `storage.rs:2124-2159`.

## Quantized matmul (`quantized.metal` GGML, `kernels/quantized.rs`)

> The GGML `quantized.metal` ships a very large kernel library (soft_max, rope, im2col, flash-attn,
> cpy, repeat, mul_mv/mul_mm for all quant types, …). Only the two matmul entry points below are
> reachable through Fuel's `call_quantized_matmul_*`; the rest are unwired dead code from Fuel's view.

### `kernel_mul_mv_<qtype>_f32` (quantized matrix-vector, transposed)
- **Op kind:** quantized GEMV `dst = W_q @ x` ("everything in reverse" GGML ne/nb param layout).
- **dtypes (qtype):** Q4_0,Q4_1,Q5_0,Q5_1,Q8_0,Q8_1,Q2K,Q3K,Q4K,Q5K,Q6K,Q8K,F16,BF16,F32.
  Output always **f32**. Per-qtype threadgroup config (nth0,nth1,align).
- **Input layouts:** lhs (activations x) offset-capable; rhs (quantized weights) zero-offset buffer;
  ne/nb metadata describes shapes/strides (nb*=0 here → contiguous assumption). dst offset-capable.
- **op params:** (b,m,n,k), ne00..ne1, nb*, r2, r3 (broadcast ratios).
- **Output:** f32 buffer, dst at `dst_offset`.
- **Source:** `metal_src/quantized.metal` (`kernel_mul_mv_*_f32`); `kernels/quantized.rs:24-176`.

### `kernel_mul_mm_<qtype>_f32` (quantized matrix-matrix, transposed)
- **Op kind:** quantized GEMM `dst = W_q @ X` (steel-style mm, 64×32 tiles, 8KB threadgroup mem).
- **dtypes (qtype):** Q4_0,Q4_1,Q5_0,Q5_1,Q8_0,Q2K,Q3K,Q4K,Q5K,Q6K,F16,BF16,F32.
  Q8_1 and Q8K are **rejected** (`UnsupportedDTypeForOp`). Output always **f32**.
- **Input layouts:** src0 (weights) with nb01/nb02/nb03 strides (offset-capable via buffer);
  src1 (X) offset-capable; full ne/nb metadata from src0/src1/dst shapes+strides. Supports batched
  via ne12/ne13 and r2/r3 broadcast ratios.
- **op params:** src0/src1/dst shapes+strides+offsets, ne00, ne02, nb01-03, ne12, nb10-13, ne0, ne1, r2, r3.
- **Output:** f32 buffer at `dst_offset`; `set_threadgroup_memory_length(0,8192)`.
- **Source:** `metal_src/quantized.metal` (`kernel_mul_mm_*_f32`); `kernels/quantized.rs:181-284`.
